# 💊 PharmaScan AI

نموذج ذكاء اصطناعي يتعرف على علب الأدوية من الصور ويعرض الاستخدام والجرعة والتحذيرات.

**EfficientNetB3 · 150 دواء · دقة 99.1% · Streamlit App**

---

## التقنيات
- **Model:** EfficientNetB3 (fine-tuned on ImageNet)
- **Dataset:** 3,900 صورة لـ 150 علبة دواء مصرية
- **Drug Info:** قاعدة بيانات محلية + OpenFDA API
- **UI:** Streamlit

## التشغيل

> للأغراض التعليمية فقط — ليس بديلاً عن استشارة الطبيب أو الصيدلاني.
